## 📌 Overview

go-repo-orchestrator is a TUI tool for Windows that helps you manage Git branches across several repositories.

Use it when you need to:
- check branches in more than one repo
- clean up old branches
- delete branches with care
- keep branch work in sync
- reduce mistakes during branch cleanup

The app uses a text-based screen, so it works in a simple terminal window. You use the keyboard to move around and choose actions.

## 🧰 How to use it safely

Use a calm, step-by-step flow:

1. Open the app.
2. Pick the repo you want to review.
3. Check the branch list.
4. Review names you do not know.
5. Mark only the branches you want to change.
6. Confirm the action before you remove anything.
7. Repeat for the next repo.

A careful review matters most when the same branch name exists in several repos.

## 🛠️ Troubleshooting

### The app does not open

Try these steps:
- make sure you downloaded the Windows file
- unzip the file if needed
- run the app again
- check that Windows did not block the file

### The window opens and closes fast

Try launching it from a terminal window so you can see messages on screen.

### No repositories appear

Check that:
- the folder you selected contains Git repos
- each repo has a `.git` folder
- the repos are not moved or renamed

### Branch data looks old

Refresh the view inside the app, then check your repo connection and local Git data.

## 📦 Suggested folder setup

You can keep your repos in one parent folder, such as:

- `C:\Users\YourName\Projects`
- `D:\Git\Repos`

A single parent folder makes setup easier because the app can scan one place for many repositories.
